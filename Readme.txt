1. Initialize the project:
$ npm install
npm will look at the dependencies that are listed in package.json and download the latest versions satisfying semver rules for all of those.

2. use $ npm install <package-name> to use if you had error or missed some packages.

3.  download bower packages and required plugins
$ gulp

4. run the project
$ gulp watch

5. run $ gem install sass if you have this error. "Gem undefined is not installed."
